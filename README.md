# 项目简介
## 核心功能
- 协同过滤算法
- 在线咨询功能
- 用户管理系统：包括系统首页、用户、音乐分类、疏导视频、音乐疗法、情感分类、心理知识、故事、心情记录、积分商品、积分兑换、交流论坛、论坛分类、举报记录、系统管理、个人中心等管理功能
- 用户互动功能：包括系统首页、疏导视频、音乐疗法、心理知识、故事、积分商品、交流论坛、通知公告、个人中心等互动功能

## 适用场景
- 心理健康领域
- 线上咨询服务
- 用户心理辅导
- 情感交流和故事分享

## 技术特点
- B/S架构，支持浏览器远程访问
- MVC设计模式，易于维护和扩展
- 使用Java、SpringBoot、Vue技术栈
- 开发环境支持IDEA、JDK1.8、Mysql、Maven、Navcat
- 提供完整的前后端代码及数据库SQL文件
